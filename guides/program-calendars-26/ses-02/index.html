Session 2: Let the Calendar Build Itself

Helping you build a program that works.

What if your unit’s program felt clear, predictable—and something scouts, volunteers, and families actually looked forward to?

This summer, Kettle Country is launching a short webinar series designed to help every pack, troop, crew, and ship build a clear, predictable program calendar that works for everyone.

Whether your unit is rebuilding, already strong, or just needs a nudge in the right direction, this series will help you:

  • Reduce leader stress
  • Build stronger participation
  • Create a program scouts want to come back to

Part of the “Guides” series for unit leaders and volunteers.
